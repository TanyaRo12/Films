Films - веб-сервис для получения подробной информации о фильмах и расписании показов.
Данный сервис решает проблему выбора фильма для просмотра, избавляя пользователя от необходимости тратить время на поиск и выбор подходящего кино.
Целевая аудитория:

⦁	обычные пользователи - люди, желающие быстро выбрать фильм для вечернего просмотра;

⦁	разработчики - могут интегрировать API в свои приложения;

⦁	владельца киносервисов - могут использовать как дополнительный функционал для своих платформ.

Перечень основных функций, выполняемых сервисом:

⦁	получение фильма. Основная функция - возвращает фильм из базы данных. Входные данные: дата и email пользователя. Выходные данные: полная информация о фильме и предложенное время просмотра.

⦁	просмотр базы фильмов. Возвращает полный список всех доступных фильмов.
